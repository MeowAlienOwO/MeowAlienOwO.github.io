Introduction

Linear regression is a typical regression task and supervised learning method. According to Andrew Ng, there we have 2 definitions:

  1. Supervised Learning: Give the dataset of right answers, the system is used to make more “correct” answers
  2. Regression: Predict continuous value output

Example: Given a set of past house price, trying to predict the price in future.

Main task

Linear regression is a system that predict the relations, and assume them as linear, with input data $\mathbf{x}$ and the result data $y$. Here, $\mathbf{x}$ is a row vector: $\mathbf{x}_d = [ x_1, x_2, x_3 … x_d]^T$.We have a learning function $f$ which gives a predicted value $f(\mathbf{x})$.
